DIFFLE

What is Diffle?

Diffle is a game inspired by Wordle without a character limit. If you enjoy Polish versions of Wordle, this game is for you.

How to play Diffle?

The goal is to guess the password using the fewest number of letters and words possible.

After each attempt, the letters will be marked with the appropriate color.
